**T# Vision** is a Turbo Vision-inspired text-mode user interface framework for modern C# and .NET.

It brings classic DOS-style windowed applications to contemporary .NET development, including windows, dialogs, menus, controls, editors, help systems, resource files, and keyboard-driven workflows.

The framework is useful for building rich console applications, developer tools, retro-style utilities, text-mode editors, and cross-platform TUI applications. It preserves familiar Turbo Vision concepts such as `TView`, `TButton`, `TDialog`, `TWindow`, and `TApplication`, while using an idiomatic C# design and pluggable rendering drivers.

T# Vision includes:

- Window, dialog, menu, control, editor, and status line infrastructure.
- Keyboard and mouse interaction patterns for text-mode applications.
- Help, history, palette, and resource file support.
- A resource system for reusable dialogs, menus, strings, palettes, and compiled resources.
- Pluggable rendering backends for different platforms and environments.
